# Vertical Agent Template Kit

A compact kit for turning a reusable role workflow into a vertical agent template with inputs, boundaries, review criteria, and launch wedge.

Inspired by: https://x.com/OpenAI/status/2061887890607755498

## What it contains
- `artifact-package/` — Obsidian package note
- `prototype/` — concrete markdown prototype
- `infographic/` — workflow explanation
- `skill-draft/` — reusable procedure draft
- `improvement-loop/` — recurring improvement tracker

## Quickstart
Read `prototype/README.md`, then fill the included template against one real workflow.
