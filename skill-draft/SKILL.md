# Vertical Agent Template Kit

Use this draft when converting a broad agent capability into a role-specific product wedge.

1. Name the buyer role.
2. Pick one frequent job-to-be-done.
3. Define required inputs and forbidden inventions.
4. Create a review surface users can trust.
5. Ship one paid workflow before adding platform features.

Pitfall: do not start with model features; start with a painful repeated role workflow.
